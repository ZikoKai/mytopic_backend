# Prompts dynamiques

Le prompt systeme impose:

- plan non statique selon le sujet,
- variation narrative (academique, business, storytelling, technique),
- variation des types de slides,
- conclusion complete (resume, synthese, ouverture).

Ajouts recents:

- renforcement de l'impact narratif (insights, implications, trade-offs),
- interdiction des doublons visuels sur une meme slide (ex: label "CONCLUSION" + titre "Conclusion"),
- consigne anti-style generique pour des sorties plus distinctives.
